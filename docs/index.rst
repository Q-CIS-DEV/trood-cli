Trood CLI
=========


General
-------
Install trood-cli.

.. code-block:: bash

    pip install trood-cli

Info
^^^^

.. code-block:: bash

    trood info

Help
^^^^

.. code-block:: bash

    trood --help

Login
^^^^^
Login in to system using your Trood account. ( Need to be registered on https://tcp.trood.com )

.. code-block:: bash

    trood login

Logout
^^^^^^

.. code-block:: bash

    trood logout
