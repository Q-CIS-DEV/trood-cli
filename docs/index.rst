Trood CLI
=========


Install
-------
Install trood-cli.

::

    pip install trood-cli

Info
^^^^

::

    $ trood info

Help
^^^^

::

    $ trood --help

Login
^^^^^
Login in to system using your Trood account. ( Need to be registered on https://tcp.trood.com )

::

    $ trood login

Logout
^^^^^^

::

    $ trood logout
